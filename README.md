# Omniauth Hatena

A Strategy to auth with Hatena in OmniAuth.

Get your ConsumerKey here:http://developer.hatena.ne.jp/ja/documents/auth/apis/oauth/consumer

## Installation

Add this line to your app's `Gemfile`:

    gem 'omniauth-hatena'

And then execute:

    $ bundle
